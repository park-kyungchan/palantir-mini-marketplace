# Overview

Apollo has a robust change management system to ensure that only high-quality, audited, and compliant changes from manual operator actions make their way to your production Environments. This section describes the types of changes that users can propose, and how the [RBAC](/docs/apollo/core/authorization/) or compliance requirements for the change target work with the built-in approval system.
