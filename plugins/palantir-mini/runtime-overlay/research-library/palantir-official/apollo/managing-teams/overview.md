# Teams

This section will walk through the settings workflows to create a new team in Apollo. This is an administrative workflow accessible only to Apollo administrators. Before reading this section, read more about how [teams](/docs/apollo/core/teams/) power [authorization](/docs/apollo/core/authorization/) in the Apollo platform.

You can access team settings through the **Hub Settings** page via the Settings icon at the bottom of the workspace navigation bar. Select the **Teams** navigation option to show several workflows available for managing teams.

![Team settings](/docs/resources/apollo/managing-teams/settings.png)

From the teams settings panel, you can:

* [Create a new team](/docs/apollo/managing-teams/create-new-team/)
* [View or edit existing teams](/docs/apollo/managing-teams/view-edit-existing-teams/)
* [Set up team contacts](/docs/apollo/managing-teams/team-contacts/)
