# Overview

The **Software Catalog** is an inventory of all [Modules](/docs/apollo/core/modules/) and Products that have been published to Apollo. Navigate to the **Modules** tab to view the available Modules.

![The Modules tab of the Software Catalog.](/docs/resources/apollo/managing-modules/software-catalog-modules.png)

Select a Module to view details such as the Entities that are included in the Module, variables that you can define when installing the Module, and the Environments where the Module has already been installed.

![The Overview tab for a Module.](/docs/resources/apollo/managing-modules/module-overview-tab.png)

Navigate to the **Contents** tab to view the configuration for each Entity in the Module.

![The Contents tab for a Module.](/docs/resources/apollo/managing-modules/module-contents-tab.png)

Navigate to the **Installations** tab to view the version, owner, date last updated, and available updates for each installation of the Module in an Environment.

![The Installations tab for a Module.](/docs/resources/apollo/managing-modules/module-installations-tab.png)

This section will walk through the workflows related to Modules, such as [creating](/docs/apollo/managing-modules/create-module/) and [installing](/docs/apollo/managing-modules/install-module/) Modules, and [updating](/docs/apollo/managing-modules/update-module-installation/) Module installations.
