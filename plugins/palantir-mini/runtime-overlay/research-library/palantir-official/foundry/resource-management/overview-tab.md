# Overview tab

The **Overview** tab in Resource Management gives users the opportunity to understand their usage at a glance, then drill into points of interest. Specifically, this tab gives an overview of:

* **Stack metrics:** Overview of activity metrics. This includes user activity, number of datasets, and number of object links.
* **Spend tracking:** Overview of spend, including usage spend over the last quarter.
* **Analysis:** Overview of usage analysis, including Foundry compute usage trends.
* **Issue identification:** Overview of usage analysis, including the top spend for usage accounts, Projects, and Ontologies.

All of these sections provide starting points for further exploration.
