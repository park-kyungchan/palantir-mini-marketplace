# Fusion

**Fusion** is a spreadsheet application for Foundry.

Fusion lets users create spreadsheets that can:

* Query Foundry datasets via search and data lookups
* Leverage cell references and spreadsheet functions
* Build interactive spreadsheets using dropdowns and linked cells
* Submit results and write back datasets to Foundry for downstream usage

If you are adding data to the Ontology, consider using [Actions](/docs/foundry/action-types/overview/).
