# Object property

The **Object property** section allows you to reference object properties inline. Add an object property widget by clicking + Add Widget or typing / in a paragraph field to open up the [widget insertion menu](/docs/foundry/notepad/embed-widgets/#from-a-document). This will open a search bar where you can find a specific object and select it to add.

By default, the `title` property of the object is shown, but you can change the displayed property in the widget property menu.

![notepad\_widgets\_object\_property](/docs/resources/foundry/notepad/notepad_widgets_object_property.png)

## Widget properties

* **Object selection:** The object to display.
* **Property:** The property to display.
* **Show icon:** Shows an object icon with the content of the property.
* **Show object properties on hover:** Opens an object information popover when hovering over the object property. The popover contains prominent object properties and a link to the object view.
* **Show property name:** Shows the property name with the content of the property.
* **Apply ontology formatting:** Applies conditional formatting as defined in the Ontology Manager.
* **Disable link:** If toggled, removes link to Object Explorer from the object property.
* **Show custom value for empty values:** If toggled, allows you to specify a custom value to show when an ontology property is empty.

## Template configuration

* **Object selection:** The object to display.

## Ontology Actions

Hover over properties to display a menu that enables editing text properties that have configured inline edits, modifying the object with Actions, and opening the object in Object Explorer.
