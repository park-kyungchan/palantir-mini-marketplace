# LaTeX widget

[LaTeX ↗](https://www.latex-project.org/about/) formatted content can be embedded using the **LaTeX** widget. You can add this content by clicking **+ Add Widget** or typing `/` in a paragraph field to open up the [widget insertion menu](/docs/foundry/notepad/embed-widgets/#from-a-document).

![Screenshot of LaTeX widget and configuration](/docs/resources/foundry/notepad/latex_widget_and_configuration.png)

## Widget properties

* **LaTeX to display:** The LaTeX expression to display.
