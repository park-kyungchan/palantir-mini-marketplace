# Resource link

Foundry resource links are created when pasting a link to another Foundry resource (for example, a Contour analysis or Code Repository) into a Notepad document. The link will automatically display the name of the resource, but keep the original path.

You can also insert a Foundry resource link by clicking + Add Widget or typing / in a paragraph field to open up the [widget insertion menu](/docs/foundry/notepad/embed-widgets/#from-a-document).

If you do not want Notepad to transform your link into a resource link, paste it using the keyboard shortcut Ctrl+Shift+V (Windows) or Cmd+Shift+V (macOS) to insert the link as plain text.
