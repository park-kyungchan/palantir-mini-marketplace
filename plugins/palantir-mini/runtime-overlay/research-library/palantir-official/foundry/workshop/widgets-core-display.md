# Core display widgets

* **[Object Table](/docs/foundry/workshop/widgets-object-table/):** Renders an object set in a tabular display with each object displayed as a row.
* **[Object List](/docs/foundry/workshop/widgets-object-list/):** Renders an object set in a list display with each object displayed as a card.
* **[Object View](/docs/foundry/workshop/widgets-object-view/):** Renders Object Explorer’s object view for a single object.
* **[Property List](/docs/foundry/workshop/widgets-property-list/):** Displays a configurable list of properties from a given object set.
* **[Links](/docs/foundry/workshop/widgets-links/):** Renders the link types and linked objects of an object set in expandable and collapsible sections.
* **[Object Set Title](/docs/foundry/workshop/widgets-object-set-title/):** Displays a summary of a given object set as a title.
* **Header text:** Renders two rows of text optimized for display in a header.
