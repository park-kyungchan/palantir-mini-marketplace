### Event-triggering / navigational widgets

* **[Button Group](/docs/foundry/workshop/widgets-button-group/):** Trigger Actions, Workshop Events, URLs, and export via buttons with configurable styles.
* **[Comments](/docs/foundry/workshop/widgets-comments/):** Enables collaboration in a Workshop module.
* **[Media Uploader](/docs/foundry/workshop/widgets-media-uploader/):** Upload media files to Foundry and trigger Actions using uploaded files as input.
* **[Tabs](/docs/foundry/workshop/widgets-tabs/):** Display tabs that can trigger Workshop Events to navigate throughout pages and layers of a module when triggered.
* **[Inline Action](/docs/foundry/workshop/widgets-inline-action-form/):** Renders an inline action form or table on an action type from the Ontology.

:::callout{theme="neutral"}
Other widgets, such as the [Metric Card](/docs/foundry/workshop/widgets-metric-card/), also support triggering commands, actions, or events through specific configuration options like the **Interactive metric** setting.
:::
