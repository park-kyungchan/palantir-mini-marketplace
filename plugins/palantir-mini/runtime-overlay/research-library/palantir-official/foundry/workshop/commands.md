# Commands

Commands in Workshop provide a common interface that you can use to declare and execute operations in other Palantir platform applications embedded in a module. Additionally, you can create [AIP Chatbots](/docs/foundry/chatbot-studio/overview/) (formerly known as AIP Agents) to execute operations that commands can read and write from other applications.

Commands can be used independently of, or in tandem with, the [App Pairing widget](/docs/foundry/cross-app-interactivity/app-pairing/) to orchestrate cross-application operations across Palantir applications.
