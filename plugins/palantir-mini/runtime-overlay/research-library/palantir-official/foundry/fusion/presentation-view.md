# Presentation view

You can preview your document in presentation view using the dropdown in the toolbar, and make it the default viewing experience using the settings button.

![presentation\_view](/docs/resources/foundry/fusion/presentation_view.png)

A nice addition to presentation view is the option to hide the default grid via the button in the toolbar.

![hide\_grid](/docs/resources/foundry/fusion/hide_grid.png)

The combination of presentation view and hiding the grid can create a very focused consumption experience for the Fusion documents you’d like to share with large audiences.

![presentation\_preview](/docs/resources/foundry/fusion/presentation_preview.png)
