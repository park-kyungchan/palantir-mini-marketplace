# Microsoft Report Builder

The Foundry integration for Microsoft Report Builder offers the following:

* Enables you to easily create reports in Report Builder backed by Foundry data.
* Offers compatibility with Foundry access controls, including granular permissions, for Desktop.
* Supports token-based authentication for Desktop.

## Supported Report Builder products

* *Report Builder Desktop* (September 2020 release or later)
