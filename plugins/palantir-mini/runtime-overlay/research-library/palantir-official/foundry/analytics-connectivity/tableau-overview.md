# Tableau

The Foundry integration for Tableau offers the following:

* Enables you to easily create Tableau interactive dashboards backed by Foundry data.
* Supports both <a target="_blank" href="https://www.tableau.com/about/blog/2016/4/tableau-online-tips-extracts-live-connections-cloud-data-53351">Extracts and Live Connections.</a>
* Compatible with Foundry access controls, including granular permissions.

## Supported Tableau products

* *Tableau Desktop* (2019.4 release or later)
* *Tableau Server* (2019.4 release or later)

:::callout{theme="neutral"}
Note that *Tableau Online via Tableau Bridge* is not currently supported by the Tableau custom connector. We anticipate that support for the connector will become available with future releases of Tableau.
:::
