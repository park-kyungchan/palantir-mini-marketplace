# Add Carbon workspace to a Marketplace product

Use [Foundry DevOps](/docs/foundry/devops/overview/) to include your Carbon workspaces in [Marketplace products](/docs/foundry/devops/core-concepts/#product) for other users to install and reuse. [Learn how to create your first product.](/docs/foundry/foundry-devops/create-products/)

## Supported features

Many Carbon features are supported with the exception of:

* [Walkthroughs](/docs/foundry/carbon/workspaces-updates/) (coming soon)
* Module types that are not Workshop, object type, or "Link to Foundry Application"
* Using groups for "Default object types for search"
* "Select objects" item type

## Adding Carbon workspaces to products

To add a Carbon workspace to a product, first [create a product](/docs/foundry/foundry-devops/create-products/), then [add outputs](/docs/foundry/foundry-devops/create-products/#add-outputs). Choose the **Add files** option to navigate to the workspace from within the [Compass](/docs/foundry/compass/overview/) filesystem and add it to your product.
