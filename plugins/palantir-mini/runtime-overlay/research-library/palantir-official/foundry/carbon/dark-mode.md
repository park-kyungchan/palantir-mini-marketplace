# Enable dark mode for an organization

Carbon Organization Administrators can set dark mode as the default theme for all workspaces in an organization. To do so, open the organization metadata configuration panel by selecting **Configure** in the **Organization** section of the edit sidebar. In the **Configure organization metadata** panel, check the box next to **Enable dark mode** and click **Apply Changes**.

<img src="./media/enable-dark-mode.png" alt="Enable dark mode" width="400" />

You can also [override the organization-level dark/light mode at the workspace level](/docs/foundry/carbon/configuration-general/#appearance).
