# Text block

> Supported in: Batch, Faster, Streaming

Insert a text description between your transformations. This does not transform the input data in any way.

**Transform categories**: Other

## Declared arguments

* **Input:** Dataset input to describe with text.<br>*Table*
* *optional* **Text:** Text description of the dataset parameter.<br>*Literal\<String>*
