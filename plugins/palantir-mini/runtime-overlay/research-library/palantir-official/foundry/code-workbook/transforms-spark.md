# Spark

[Apache Spark ↗](https://spark.apache.org/docs/latest/index.html) is a commonly-used open-source engine used for data engineering, data science, and machine learning workflows on large-scale data.

Code Workbook natively supports accessing data in Spark across all languages. For detailed information, refer to the following pages:
