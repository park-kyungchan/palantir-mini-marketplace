# Upgrade Assistant

Upgrade Assistant provides visibility on upcoming [platform changes](/docs/foundry/upgrade-assistant/platform-changes/). Platform changes such as upgrades can, in some cases, cause issues with existing workflows.

To help adapt to platform changes, Upgrade Assistant provides a [list of resources](/docs/foundry/upgrade-assistant/impacted-resources/) that were identified as being affected by a particular platform upgrade. Each upgrade also comes with instructions on how to remediate the affected issues.

To ensure that action is taken, each resource is assigned to users and groups. Upgrade Assistant determines resource assignees based on various [rules and heuristics](/docs/foundry/upgrade-assistant/resource-assignment/).

As the deadline for an upgrade approaches, Upgrade Assistant will send [notifications](/docs/foundry/upgrade-assistant/notifications/) to users who have been assigned resources as a reminder to take the actions required to prevent workflow disruption.
