# Custom plot (transform table)

Create a new time series column in the transform table where you can customize each plot's value, color, title, x-axis range, and y-axis range using column values.

## Input type

Time series

## Output type

Time series

## Usage information

| Functionality | Availability |
| --- | --- |
| [Standard Quiver card](/docs/foundry/quiver/core-concepts/#cards) | Unsupported |
| [Transform table transform](/docs/foundry/quiver/cards-transform-table/) | Supported |
