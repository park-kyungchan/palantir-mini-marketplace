# Numerical scatter plot

Scatter plot one or more object sets of the same type by plotting two numerical properties against one another.

:::callout{theme="warning"}
A dot will be drawn for each object in each object set. This may cause performance issues for object sets with more than 50,000 objects.
:::

## Input type

Object set

## Output type

Object selection
