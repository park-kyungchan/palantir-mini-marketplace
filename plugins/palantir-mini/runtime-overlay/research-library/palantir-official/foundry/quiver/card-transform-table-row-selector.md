# Transform table row selector

Creates select parameter for selecting a single row from a transform table.  The row selection can be defined either through a primary key or by a row index (starting from 0).

## Input type

Transform table

## Output type

Transform table

## Usage information

| Functionality | Availability |
| --- | --- |
| [Standard Quiver card](/docs/foundry/quiver/core-concepts/#cards) | Supported |
| [Transform table transform](/docs/foundry/quiver/cards-transform-table/) | Unsupported |
