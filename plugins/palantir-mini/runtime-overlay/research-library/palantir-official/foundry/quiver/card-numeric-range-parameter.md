# Numeric range parameter

Create a numeric range input field which can be used as a variable in other cards that take numeric range variables as input.

Numeric ranges are often used to define y-axis ranges on time series charts. [Learn more about time series chart ranges.](/docs/foundry/quiver/timeseries-ranges/)

## Input type

Flow start

## Output type

Number range
