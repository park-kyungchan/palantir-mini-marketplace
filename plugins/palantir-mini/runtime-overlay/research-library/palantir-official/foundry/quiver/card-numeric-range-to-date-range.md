# Numeric range to date range

Convert a numeric range into a date range. For conversion, choose to use Unix milliseconds or Unix seconds.

This card is similar to the [number to date](/docs/foundry/quiver/card-number-to-date/) card, but it operates on a range rather than a single value.

## Input type

Number range

## Output type

Time range
