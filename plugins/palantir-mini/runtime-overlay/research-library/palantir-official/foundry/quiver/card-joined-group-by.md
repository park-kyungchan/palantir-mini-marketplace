# Joined group by

Performs a [group by](/docs/foundry/quiver/card-group-by/) operation, then joins the result onto the current transform table.  This can be useful for comparison workflows, such as comparing each row against all rows in the specified group.

## Input type

Transform table

## Output type

Transform table

## Usage information

| Functionality | Availability |
| --- | --- |
| [Standard Quiver card](/docs/foundry/quiver/core-concepts/#cards) | Unsupported |
| [Transform table transform](/docs/foundry/quiver/cards-transform-table/) | Supported |
