# Create range

Create a new column in the transform table with a specified range value that is the same for all rows. Can be useful for hooking a range parameter's value into the transform table.

## Input type

Flow start, time, number

## Output type

Number range, time range

## Usage information

| Functionality | Availability |
| --- | --- |
| [Standard Quiver card](/docs/foundry/quiver/core-concepts/#cards) | Unsupported |
| [Transform table transform](/docs/foundry/quiver/cards-transform-table/) | Supported |
