# Visual function property select

Uses a [visual function](/docs/foundry/quiver/visual-functions-overview/) to save repeatable logic that returns a string selection of object property values.

## Input type

Object set, single object, number, string, time, boolean

## Output type

String array

## Usage information

| Functionality | Availability |
| --- | --- |
| [Standard Quiver card](/docs/foundry/quiver/core-concepts/#cards) | Supported |
| [Transform table transform](/docs/foundry/quiver/cards-transform-table/) | Unsupported |
