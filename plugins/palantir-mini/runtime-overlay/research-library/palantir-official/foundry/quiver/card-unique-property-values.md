# Unique property values

Returns the set of unique values for a low cardinality, string property over an object set. This can also be used for filtering. Note that only the top 1,000 values will be used.

## Input type

Object set

## Output type

String, string array

## Usage information

| Functionality | Availability |
| --- | --- |
| [Standard Quiver card](/docs/foundry/quiver/core-concepts/#cards) | Supported |
| [Transform table transform](/docs/foundry/quiver/cards-transform-table/) | Unsupported |
