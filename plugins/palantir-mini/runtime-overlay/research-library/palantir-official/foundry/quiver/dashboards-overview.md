# Dashboards

Quiver dashboard mode allows you to present insights from your analysis in read-only, interactive dashboards. Dashboards can be used as standalone views or embedded in other Foundry applications.

## Key features

* Create multiple dashboards per analysis.
* Start from a blank dashboard and easily drag and drop content from your analysis into the dashboard.
* Choose between three view styles to optimize your dashboard for standalone consumption or for embedding in other applications.
* Embed dashboards in other parts of Foundry, including [Workshop modules](/docs/foundry/quiver/dashboards-workshop/), [Notepad documents](/docs/foundry/quiver/dashboards-notepad/), [Carbon workspaces](/docs/foundry/quiver/dashboards-carbon/), and [Object Views](/docs/foundry/quiver/dashboards-object-view/).
