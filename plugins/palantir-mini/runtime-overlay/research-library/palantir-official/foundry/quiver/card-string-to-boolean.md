# String to Boolean

Convert literal true/false string values to their corresponding boolean values, with all other strings set to `false`. Optionally, set the default boolean value to`true` and/or specify which string values will be mapped to `true` and/or which to `false`."

## Input type

String

## Output type

Boolean
