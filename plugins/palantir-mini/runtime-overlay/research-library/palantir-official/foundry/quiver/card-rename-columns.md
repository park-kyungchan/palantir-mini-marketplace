# Rename columns

Rename the display names for one or more columns in a transform table.  Note that this transform will only update a column's display name; the transform will not change the column's underlying column ID.

## Input type

Transform table

## Output type

Transform table

## Usage information

| Functionality | Availability |
| --- | --- |
| [Standard Quiver card](/docs/foundry/quiver/core-concepts/#cards) | Unsupported |
| [Transform table transform](/docs/foundry/quiver/cards-transform-table/) | Supported |
