# Property value select parameter

Creates a multiselect (or single select) parameter based on values from an object set property. Property value select parameters can be used to parameterize filters on low cardinality string properties in object sets, such as categories.

## Input type

Object set

## Output type

String, string array

## Usage information

| Functionality | Availability |
| --- | --- |
| [Standard Quiver card](/docs/foundry/quiver/core-concepts/#cards) | Supported |
| [Transform table transform](/docs/foundry/quiver/cards-transform-table/) | Unsupported |
