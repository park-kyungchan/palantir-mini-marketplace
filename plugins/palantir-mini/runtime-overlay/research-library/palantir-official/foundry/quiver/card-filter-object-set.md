# Filter object set

Takes as input an object set card and returns a (filtered) object set based on a set of logical conditions you can define on both the objects properties and the properties of linked objects. Learn more about [filtering object sets](/docs/foundry/quiver/objects-filter/).

## Input type

Object set

## Output type

Object set
