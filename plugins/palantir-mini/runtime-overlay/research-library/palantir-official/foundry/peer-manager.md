# Peer Manager

The Peer Manager application enables you to view and monitor jobs associated with established peering connections, which provide the ability to import or export [objects](/docs/foundry/object-link-types/object-types-overview/) between Foundry enrollments.

![Peer Manager's homepage displays a table of peer connections.](/docs/resources/foundry/peer-manager/pma_home.png)

## Features

* **Create peer connections:** Create or import [peer connection invite codes](/docs/foundry/peer-manager/create-a-peer-connection/) to create a peer connection from your enrollment to another enrollment.
* **View peer connections:** Select a connection to launch the application's **Overview** window, where Peer Manager displays the data type, source, and directionality of peering connections associated with your [spaces](/docs/foundry/security/orgs-and-spaces/#spaces).
* **Configure ontology peering:** Synchronize ontology data across peer connections through [Ontology peering](/docs/foundry/peer-manager/ontology-peering-overview/).
* **Configure file peering:** Synchronize Gotham files across peer connections through [file peering](/docs/foundry/peer-manager/configure-file-peering/).
* **Monitor peering jobs:** Track the health of your peer connections by viewing the status of individual peering jobs.
