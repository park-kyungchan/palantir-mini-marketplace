# Workshop (Community)

### Is it possible for users to view how a Workshop application is configured without editing it?

To view the configuration of a Workshop application without editing it, users would need to make a copy of the application. A duplicate of the application can be created by replacing `/view/latest` in the application URL with `/duplicate`.
