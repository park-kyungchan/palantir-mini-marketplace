# Foundry Metadata (Community)

### Is it possible to find version numbers of some Foundry services from within Foundry or using Foundry APIs?

It is not possible to find version numbers of Foundry services from within Foundry or by hitting APIs - this is not something exposed to end-users. Version numbers of underlying micro-services are implicitly treated as implementation details and are not exposed to end-users which is a commonly followed paradigm in SaaS offerings.
