# Notifications (Community)

### What troubleshooting steps can be taken if users don't receive emails from Foundry?

If users don't receive any emails from Foundry, an Organization Administrator can review the email allowlist to ensure the target domain is on the allow list. More information can be found [here](/docs/foundry/email/email-allowlisting/).
