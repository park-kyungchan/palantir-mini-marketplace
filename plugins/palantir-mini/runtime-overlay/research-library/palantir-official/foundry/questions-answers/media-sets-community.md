# Media sets (Community)

### Is there a way to stop requiring an organization marking on a media set?

To stop propagation of security and organization markings, you can use the `stop_propagating` and `stop_requiring` properties on media set inputs as shown [here](/docs/foundry/platform-security-management/manage-markings/#remove-an-inherited-marking).
