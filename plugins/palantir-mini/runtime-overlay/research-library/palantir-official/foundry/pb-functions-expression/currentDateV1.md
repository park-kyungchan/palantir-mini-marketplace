# Current date

> Supported in: Batch, Faster, Streaming

Returns the current date of when computation started.

**Expression categories:** Datetime

## Declared arguments

* *optional* **Time zone:** The time zone to use when computing the current date. Default is UTC.<br>*TimeZone*

**Output type:** *Date*
