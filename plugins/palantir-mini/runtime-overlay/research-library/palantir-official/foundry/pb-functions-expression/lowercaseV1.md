# Lowercase

> Supported in: Batch, Faster, Streaming

Converts all characters in string to lowercase.

**Expression categories:** String

## Declared arguments

* **Expression:** Expression to convert to lowercase.<br>*Expression\<String>*

**Output type:** *String*

## Examples

### Example 1: Base case

**Argument values:**

* **Expression:** Hello World

**Output:** hello world

***

### Example 2: Null case

**Argument values:**

* **Expression:** *null*

**Output:** *null*
