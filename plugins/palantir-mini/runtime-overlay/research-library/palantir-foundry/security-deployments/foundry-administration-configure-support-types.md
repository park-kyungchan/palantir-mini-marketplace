Configure support types
=======================

When using the [simplified issue filing flow](/docs/foundry/getting-help/issues/#simplified-flow), custom support types allow you to add support types that are specific to your workflows and operating model. This makes it easier for users to file tickets and reduces the risk of tickets going to the incorrect assignee.

To configure custom support types, open Control Panel and navigate to **Enrollment settings > Support > Support types** and select **Add**.

![Support type creation form](/docs/resources/foundry/administration/support-type-creation.png?width=600)

When the [simplified issue filing flow](/docs/foundry/getting-help/issues/#simplified-flow) is enabled, but support types have not been configured, users will be unable to file issues even if they have access to the Foundry Issues application.

![Simplified issue filing flow without support types.](/docs/resources/foundry/administration/no-support-types.png?width=600)
