AIP observability
==================

AIP observability refers to a set of capabilities in Workflow Lineage that provide visibility into your AIP and Ontology workflow executions, including metrics, execution history, distributed tracing, logging, and log search. These capabilities are part of Palantir's platform-wide observability strategy.

To learn more about available tools and how to get started, see the AIP observability section at `/docs/foundry/aip-observability/overview/`.
