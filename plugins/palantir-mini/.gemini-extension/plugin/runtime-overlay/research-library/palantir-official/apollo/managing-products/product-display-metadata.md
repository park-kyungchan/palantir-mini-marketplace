# Edit Product display metadata

In the **Display metadata** section of the Product settings, you can update the display name, description, and README for the Product.

The display name will replace the Product name in the Product page header, as well as anywhere else in Apollo that the Product is referenced, such as dropdowns and lists.

The description should include high level information about the Product, and will be displayed under the title in the Product page header.

<img alt="Example display name, description, and README." src="./media/display-metadata-example.png" width=700>

The README field should include detailed information about the Product, such as documentation or upgrade instructions. The README supports Markdown rendering, and will render external links, tables, lists, and more. You can view the README in the Product  **Releases** tab.
