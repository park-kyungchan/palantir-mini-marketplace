# Formulas

Fusion supports standard spreadsheet formulas (cell references, addition, etc.) as well as functions. Fusion’s function library is a combination of Contour expression language DSL and Fusion specific spreadsheet functions. For a list of functions, visit the [function library](/docs/foundry/fusion/function-library/).

Type `=` into any cell to use a formula. Typing `=a` into the formula bar will give you a list of suggestions.
