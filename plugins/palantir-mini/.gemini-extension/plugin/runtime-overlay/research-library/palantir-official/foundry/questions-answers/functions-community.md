# Functions (Community)

### Does the Usage History in Functions also cover Actions?

The Usage History section attributes executions to the resources which initiated the execution, so generally that will be applications like Workshop or Quiver (instead of the Action within the Workshop for example).

There is currently no way to view all of the Actions that a given Function backs.
