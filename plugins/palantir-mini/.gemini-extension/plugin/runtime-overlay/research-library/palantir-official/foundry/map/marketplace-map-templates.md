# Add Map templates to a Marketplace product

Use [Foundry DevOps](/docs/foundry/devops/overview/) to include your [Map templates](/docs/foundry/map/templates/) in [Marketplace products](/docs/foundry/devops/core-concepts/#product) for other users to install and reuse. [Learn how to create your first product.](/docs/foundry/foundry-devops/create-products/)

## Supported features

All Map template features are supported with the exception of [Map Layers](/docs/foundry/map/layer-editor/).

## Adding Map templates to products

To add a Map template to a product, first [create a product](/docs/foundry/foundry-devops/create-products/), then [add outputs](/docs/foundry/foundry-devops/create-products/#add-outputs). Choose the **Add files** option to navigate to the Map template from within the [Compass](/docs/foundry/compass/overview/) filesystem and add it to your product.
