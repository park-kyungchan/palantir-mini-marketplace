# Cover pages

The Project **Cover Page** section offers a Markdown-based rich-text editor for writing comprehensive documentation about the Project.

Project owners are encouraged to add documentation to describe the Project's goals, as well as to guide users toward relevant files and active areas of work. This section can also provide instructions on which groups new users should request access to. Cover pages can also be used as user guides for applications contained in the Project.

Cover pages can be configured by Project owners to be discoverable by all users in the Project's organization, even in cases when a Project has markings applied to it. Users without access to the Project or its files can still discover and view the Project's cover page. This can be useful for Projects containing sensitive data and applications, as the discoverable cover page enables relevant users to find the Project and request access if appropriate.
