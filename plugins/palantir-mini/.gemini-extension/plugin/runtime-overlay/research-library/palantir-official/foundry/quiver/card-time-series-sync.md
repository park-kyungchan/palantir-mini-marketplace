# Time series sync

Load a time series directly from a [time series sync](/docs/foundry/time-series/time-series-concepts-glossary/#time-series-sync). To add additional series, click the '+' icon next to the sync name in the chart legend, add this card again, or duplicate the plot.

## Input type

Flow start

## Output type

Time series
