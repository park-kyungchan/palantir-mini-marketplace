# Sensitive Data Scanner

Sensitive Data Scanner (SDS, formerly known as "Foundry Inference") helps organizations discover and secure sensitive data within Foundry. Governance users can use SDS to specify patterns of sensitive data to identify, along with the appropriate automated action to take when such sensitive data is found.

![Screenshot of Sensitive Data Scanner](/docs/resources/foundry/sensitive-data-scanner/sds_landing_page.png)

Based on these instructions, Sensitive Data Scanner can either run a one-time scan or scheduled recurring scans across all specified datasets. Sensitive Data Scanner gives governance users powerful insight and control over their data in Foundry and simplifies compliance with data protection regulations across the world.

We recommend new users review the [core concepts](/docs/foundry/sensitive-data-scanner/core-concepts/) for Sensitive Data Scanner and the instructions on how to [create an Sensitive Data Scan](/docs/foundry/sensitive-data-scanner/create-a-sensitive-data-scan/). Review the [SDS best practices guide](/docs/foundry/sensitive-data-scanner/best-practices/) to reduce scan compute time, boost workflow efficiency, and manage user interactions based on resource permissions.

:::callout{theme="neutral"}
For other functionality in Foundry that can help your organization comply with GDPR, CCPA, LGPD and other privacy laws, see our page on [data protection and governance](/docs/foundry/security/data-protection-and-governance/).
:::

Sensitive Data Scanner may not be enabled by default on your enrollment. To enable the feature, follow the instructions below:

1. Navigate to Control Panel.
2. From the left sidebar, choose your Organization.
3. Select **Application access**.
4. Scroll down to find **Sensitive Data Scanner** under the **Security and governance** section, then select the **Manage** option.

![Sensitive Data Scanner feature under the Security and governance section of Control Panel.](/docs/resources/foundry/sensitive-data-scanner/control-panel.png)
