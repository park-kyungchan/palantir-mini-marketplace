# Charts cards

Back to: [Index of cards](/docs/foundry/quiver/cards-index/)

Cards in this section are used to visualize data. Some of these cards aggregate (“group by”) the data by category, and the output aggregations can be used as inputs to other cards. Review the documentation on [creating and configuring charts](/docs/foundry/quiver/objects-chart-drilldown/) for more information.

The following cards accept object sets as an input:

* [Bar chart](/docs/foundry/quiver/card-bar-chart/)
* [Line chart](/docs/foundry/quiver/card-line-chart/)
* [Pie chart](/docs/foundry/quiver/card-pie-chart/)
* [Categorical scatter plot](/docs/foundry/quiver/card-categorical-scatter-plot/)
* [Numerical scatter plot](/docs/foundry/quiver/card-numerical-scatter-plot/)
* [Heat grid](/docs/foundry/quiver/card-heat-grid/)
* [Map](/docs/foundry/quiver/card-map/)
* [Waterfall plot](/docs/foundry/quiver/card-waterfall-plot/)
* [Code function categorical plot](/docs/foundry/quiver/card-function-on-objects-plot/)
* [Correlation matrix](/docs/foundry/quiver/card-correlation-matrix/)
* [Events timeline](/docs/foundry/quiver/cards-index-event-sets/#events-timeline)

The following cards accept single objects as an input:

* [Media property](/docs/foundry/quiver/card-media-property/)

The following cards accept transform tables as an input:

* [Categorical plot from transform table](/docs/foundry/quiver/card-transform-table-plot/)
* [Vega plot](/docs/foundry/quiver/card-vega-plot/)

The following cards accept materializations as an input:

* [Categorical plot from materialization](/docs/foundry/quiver/card-categorical-plot-materialization/)

The following cards accept categorical charts as an input:

* [Categorical formula plot](/docs/foundry/quiver/card-categorical-formula-plot/)
* [Overlay chart](/docs/foundry/quiver/card-overlay-chart/)
* [Segment formula plot](/docs/foundry/quiver/card-segment-formula-plot/)

The following cards accept time series as an input:

* [Scatter plot regression](/docs/foundry/quiver/card-scatter-plot-regression/)
* [Time series distribution](/docs/foundry/quiver/card-time-series-distribution/)
* [Time series heat grid](/docs/foundry/quiver/card-time-series-heat-grid/)
* [Time series scatter plot](/docs/foundry/quiver/card-scatter-plot-regression/)
