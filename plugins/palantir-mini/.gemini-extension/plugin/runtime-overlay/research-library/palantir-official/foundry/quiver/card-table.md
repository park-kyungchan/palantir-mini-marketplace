# Table

Show chosen properties of your chosen objects as rows of a table and use this to manually select certain objects for drill-down if desired. Tables can also include the properties of linked objects and time series (time series properties or linked sensors) of the object displayed as sparklines.

Selections can also be made from the table. Selecting a single object will drill into that specific object. This can be useful, since some plots only take single objects as input.  Holding `Cmd` (macOS) or `Ctrl` (Windows) when selecting an object will allow multi selection.

## Input type

Object set

## Output type

Object selection
