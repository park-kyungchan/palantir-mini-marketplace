# Anchor link

Notepad allows you to create anchor links to reference particular pieces of a document. Sharing this link with other users ensures that they will be taken to this specific part of the Notepad document upon accessing the link. Similarly, you can also use anchor links in larger documents to create a custom outline.

To copy an anchor link, move your cursor to the content you want to reference and select **Copy anchor link** in the drag handle menu on the right. You can alternatively select the content first and then select the anchor icon in the toolbar. You can then paste your link to share it with other users.

![notepad\_widgets\_anchor\_link](/docs/resources/foundry/notepad/notepad_widgets_anchor_link.png)

## Display text

Anchor links pasted into the same document they are copied from will display either the text or widget type of the copied element. Anchor links pasted into a separate document will also show the title of the source document. All anchor links will display their source document RID in a tooltip.
