# Events and time series

Vertex includes a suite of features designed to help you understand how your organization is changing over time. This can be helpful for analyzing events in the past and forecasting events in the future, especially in combination with Vertex's support for [scenarios](/docs/foundry/vertex/scenarios-overview/).

There are two main types of temporal data you should be familiar with: **events** and **time series**.

* Events are [object types](/docs/foundry/object-link-types/object-types-overview/) configured in the Ontology that include temporal information—minimally, two timestamps that indicate the start and end of the event.
* [Time series](/docs/foundry/time-series/time-series-overview/) represent measured values over time and can be configured in the ontology as [time series properties](/docs/foundry/time-series/time-series-setup/).
