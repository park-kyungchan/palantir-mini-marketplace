# Action button

Create a button that uses an [ontology action](/docs/foundry/action-types/overview/) to write data back to the Ontology.  Action buttons can be added to the parameters panel, canvas, time series charts, and dashboards to allow users to write data back to the Ontology; for example, by creating objects, updating properties on existing objects, or modifying object links.

[Learn more about how action buttons can be used in Quiver.](/docs/foundry/quiver/cards-index-writeback/)

## Input type

Object set, single object

## Output type

Flow end
