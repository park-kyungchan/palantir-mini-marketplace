# SQL transforms

**SQL** is the easiest language to use when authoring data transformations in Foundry, while enabling a broad range of advanced data manipulation patterns thanks to the expressiveness of Spark SQL, including filtering, aggregations, derivations, and window functions.

Get started with SQL transforms using the [simple batch pipeline tutorial](/docs/foundry/building-pipelines/create-batch-pipeline-cr/) for Code Repositories, or explore the [Spark SQL reference](/docs/foundry/transforms-sql/spark-reference/). Note that Foundry does not support incremental transforms in SQL.
