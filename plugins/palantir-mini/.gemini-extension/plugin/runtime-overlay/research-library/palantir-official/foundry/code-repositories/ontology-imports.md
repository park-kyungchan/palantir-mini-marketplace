# Ontology imports

The **Ontology** tab in Code Repositories is used to import [object and link types](/docs/foundry/ontology/core-concepts/) for use in [Functions on Objects](/docs/foundry/functions/functions-on-objects/). Refer to the Functions documentation on [ontology imports](/docs/foundry/functions/ontology-imports/) to learn more.
