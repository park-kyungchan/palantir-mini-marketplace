# Numeric parameter

Create a numeric input field which can be used as a variable in other cards that take a number as input.

Examples include:

* Numeric properties filter values in the [filter object set](/docs/foundry/quiver/card-filter-object-set/) card.
* Inputs in [formula cards](/docs/foundry/quiver/cards-formulas/), which can be useful as a way to quickly test different formula coefficients.
* Window duration value in the [rolling aggregate](/docs/foundry/quiver/card-rolling-aggregate/) card.

## Input type

Flow start

## Output type

Number
