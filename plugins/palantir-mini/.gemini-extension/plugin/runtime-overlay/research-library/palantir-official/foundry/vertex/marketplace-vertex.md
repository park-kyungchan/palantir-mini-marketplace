# Add Vertex graph template to a Marketplace product

Use [Foundry DevOps](/docs/foundry/devops/overview/) to include your [Vertex graph templates](/docs/foundry/vertex/graphs-template/) in [Marketplace products](/docs/foundry/devops/core-concepts/#product) for other users to install and reuse. [Learn how to create your first product.](/docs/foundry/foundry-devops/create-products/)

## Supported features

All Vertex graph template features are supported with the exception of saved Search Arounds.

## Adding Vertex graph templates to products

To add a Vertex graph template to a product, first [create a product](/docs/foundry/foundry-devops/create-products/), then [add outputs](/docs/foundry/foundry-devops/create-products/#add-outputs). Choose the **Add files** option to navigate to the graph template from within the [Compass](/docs/foundry/compass/overview/) filesystem and add it to your product.
