# Ontology MCP (OMCP)

Ontology MCP is a Developer Console feature that exposes your application's ontology resources as Model Context Protocol (MCP) tools. This enables AI agents and external systems to interact with your ontology as MCP clients. To learn more about Ontology MCP, refer to the [documentation](/docs/foundry/ontology-mcp/overview/).
