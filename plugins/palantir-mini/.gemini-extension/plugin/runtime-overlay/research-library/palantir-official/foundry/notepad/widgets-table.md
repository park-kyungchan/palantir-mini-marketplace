# Table

Notepad offers a native **Table** widget. You can add one by selecting **+ Add Widget** or by using the / keyboard shortcut in a paragraph field to open the [widget insertion menu](/docs/foundry/notepad/embed-widgets/#from-a-document). You can configure your table's dimensions using the sliders in the menu.

When you select the newly inserted table, a specific **Table Operations** toolbar will appear in the Notepad toolbar. The **Table Operations** toolbar allows you to insert or remove additional rows and columns, set cell background colors, or switch between default and dense display modes. Delete your selected table by clicking on the bin icon. Alternatively, you can use the right-click menu to insert or remove rows and columns.

![The table operations toolbar or right-click menu can be used to edit a table in Notepad.](/docs/resources/foundry/notepad/notepad_widgets_table.png)

**Table row generator**

You can dynamically generate table rows based on each object in an object set using the table row generator. Learn more in the [table row generator documentation](/docs/foundry/notepad/widgets-table-row-generator/).
