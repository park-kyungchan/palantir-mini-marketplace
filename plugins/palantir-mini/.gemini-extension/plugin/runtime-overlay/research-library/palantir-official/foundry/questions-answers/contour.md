# Contour

### Is there a URL parameter that can be added to open a Contour dashboard in full screen mode and minimize the parameters pane?

There is no URL parameter to automatically minimize the parameters pane upon opening a Contour dashboard, but you can copy a link while the dashboard is in full screen mode that will open the dashboard in full screen when navigating to it.

*Timestamp:* April 16, 2024

### Does Contour currently have version control or history features?

No, Contour currently does not have any version control features. If you need to perform analyses with versioning and revision history, we recommend using [Code Workbook](/docs/foundry/code-workbook/overview/).

*Timestamp:* April 16, 2024

### How can I create a new column that calculates the number of days between two date columns?

To calculate the number of days between two date columns, use the `date_diff` function in an expression board.

*Timestamp:* April 16, 2024
