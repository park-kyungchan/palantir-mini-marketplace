# Grouped time series plot

The grouped time series plot card enables you to visualize multiple time series together as part of the same time series plot. For a comprehensive guide on batch time series analysis, including other visualization and analysis approaches, see [batch analyze time series](/docs/foundry/quiver/timeseries-batch-analyze/).
