# Models

You can create and iterate on Foundry Model Assets in [Code Repositories](/docs/foundry/integrate-models/model-asset-code-repositories/) and [Code Workspaces](/docs/foundry/code-workspaces/overview/). A Foundry Model is a "container for logic" - an object which encodes knowledge about your data to create predictions and empower decisions. Models can be saved for later use, providing lineage, history, security, and branching.

![Screenshot of Model Library](/docs/resources/foundry/analytics/models-example-model-library.png)

Learn more about [creating models in Code Repositories](/docs/foundry/model-integration/tutorial-intro/), [creating models in Code Workspaces](/docs/foundry/code-workspaces/getting-started/), and [machine learning and modeling in Foundry](/docs/foundry/model-integration/overview/).
