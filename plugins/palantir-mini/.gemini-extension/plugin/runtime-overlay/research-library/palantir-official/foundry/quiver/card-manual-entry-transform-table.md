# Manual entry transform table

Allows a user to manually enter up to 5,000 rows of data to create a new transform table. The manual entry transform table card has a spreadsheet-like user interface and supports five data types: string, number, boolean, time, and [time series](/docs/foundry/quiver/timeseries-overview/). Users can interact with and apply operations on manual entry transform tables the same way as other transform tables.

See [Transform tables - Input: Manual entry transform tables](/docs/foundry/quiver/cards-transform-table/#input-manual-entry-transform-tables) for more information.

## Input type

Flow start

## Output type

Transform table
