# Duration unit parameter

Create a duration unit input field which can be used as a variable in other cards that take a duration as input. This can be used in many of the time series plots such as controlling the window size in the [rolling aggregate](/docs/foundry/quiver/card-rolling-aggregate/) plot, or controlling the shift duration in the [shift time series](/docs/foundry/quiver/card-shift-time-series/) plot.

## Input type

Flow start

## Output type

Duration unit
