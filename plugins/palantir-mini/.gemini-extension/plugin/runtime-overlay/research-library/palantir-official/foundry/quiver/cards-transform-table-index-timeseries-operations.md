# Time series operations

Back to: [Index of transforms](/docs/foundry/quiver/cards-transform-table-index/)

The transform table enables batch analysis of time series data. For a comprehensive guide on batch time series analysis, including workflows and best practices, see [batch analyze time series](/docs/foundry/quiver/timeseries-batch-analyze/).

The following time series operations are available in transform tables:

Transforms in this section operate on a [time series](/docs/foundry/quiver/timeseries-overview/).
Learn more about [how to search for and add a transformation](/docs/foundry/quiver/cards-transform-table/#add-a-transformation).
