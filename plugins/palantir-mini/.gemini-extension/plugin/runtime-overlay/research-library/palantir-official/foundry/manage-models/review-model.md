# Review models

**Reviews** on models allow various stakeholders from different backgrounds to collaborate on model evaluation, and set up processes for holistic release decisions. Stakeholders may include modeling peers, modeling leads, organizational owners, subject-matter experts (SMEs), internal governance or ethics boards, and more.

Objective owners can set [roles](/docs/foundry/security/projects-and-roles/#roles) to control access for review, release, and deployment.
