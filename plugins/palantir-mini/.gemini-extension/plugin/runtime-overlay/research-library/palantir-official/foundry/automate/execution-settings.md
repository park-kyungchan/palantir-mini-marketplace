# Automation settings

Automation settings are divided into two main categories for conditions and effects; follow the links below for more information about each category.

* **[Condition settings](/docs/foundry/automate/condition-settings/):** Settings that control how conditions are evaluated and how automation events are processed.
* **[Effect settings](/docs/foundry/automate/effect-settings/):** Settings that control how effects are executed.
