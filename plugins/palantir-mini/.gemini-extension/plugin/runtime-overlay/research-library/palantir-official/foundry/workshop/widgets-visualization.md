# Visualization widgets

* **[Chart: Pie](/docs/foundry/workshop/widgets-pie-chart/):** Visualize object data as a pie or donut chart.

* **[Chart: XY](/docs/foundry/workshop/widgets-chart/):** Visualize an object set as a configurable bar, line, or scatter chart with support for multiple series, axes, and styling options.

* **[Chart: Vega](/docs/foundry/workshop/widgets-vega-chart/):** Create fully customizable charts using the Vega and Vega-Lite grammars.

* **[Map](/docs/foundry/workshop/widgets-map/):** Display an object set as a configurable, interactive geospatial visualization.

* **[Free-form Analysis](/docs/foundry/workshop/widget-free-form-analysis/):** Enable users to independently investigate object data with flexibility, within the framework of the Workshop application.

* **[Gantt Chart](/docs/foundry/workshop/widgets-gantt-chart/):** Display objects with time properties in a horizontal timeline view.

* **[Image Annotation](/docs/foundry/workshop/widgets-image-annotation/):** Annotate images by drawing rectangles around areas of interest.

* **[Linked Compass Resources](/docs/foundry/workshop/widgets-linked-compass-resources/):** Renders Compass resources linked to an object.

* **[Markdown](/docs/foundry/workshop/widgets-markdown/):** Displays formatted text and can optionally configure object references in text.

* **[Media Preview](/docs/foundry/workshop/widgets-media-preview/):** Given a URL, Foundry RID, or Base64-encoded string, render a preview of the specified media (for example: PNG, JPEG, or PDF). You can also display visualizations returned by Functions that are represented as base64 strings.

* **[Metric Card](/docs/foundry/workshop/widgets-metric-card/):** Renders a configurable card to highlight key metrics.

* **[PDF Viewer](/docs/foundry/workshop/widgets-pdf-viewer/):** Render PDFs (portable document formats) with keyword search capabilities.

* **[Pivot Table](/docs/foundry/workshop/widgets-pivot-table/):** Enables the dynamic grouping and aggregation of object data and then displays this aggregated data in tabular form.

* **[Resource List](/docs/foundry/workshop/widgets-resource-list/):** Display various types of Foundry resources.

* **Status Tracker:** Display statuses of a process on a timeline.

* **[Stepper](/docs/foundry/workshop/widgets-stepper/):** Track progress of a task through a set of steps completed both linearly or non-linearly.

* **[Timeline](/docs/foundry/workshop/widgets-timeline/):** Visualize temporal data, rendering objects as events a timeline.

* **Chart: Waterfall:** Visualize object data as a waterfall chart.

* **[Action Log Timeline](/docs/foundry/workshop/widgets-action-log-timeline/):** Renders object action logs in a temporal view.
