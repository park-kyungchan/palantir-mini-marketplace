# Duplicate a report

:::callout{theme="warning"}
The Foundry Reports application has been sunsetted, and legacy reports are now available as view-only. Refer to the [Reports sunset announcement](/docs/foundry/reports/overview/) for further details.
:::

There are several ways to duplicate a report; each is described below.

## Via the Report

1. Open the **Actions** menu.
2. Click **Duplicate report**

## Via the Application header

1. Open the **File** menu.
2. Click **Duplicate file**

## Via Your files view

As with any resource in Palantir Foundry—whether a report or something else—you can duplicate a report while viewing the resources in a Project:

1. Right click on the report in your list of files.
2. Click **Actions** in the context menu that appears.
3. Click **Duplicate report**.
