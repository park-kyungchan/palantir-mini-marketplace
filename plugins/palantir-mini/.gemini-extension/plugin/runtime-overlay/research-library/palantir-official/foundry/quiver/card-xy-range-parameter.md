# X/Y range parameter

Creates a parameter that defines an X range and a Y range. Each range can either be a time range, numeric range, or ordinal (string) range. This is primarily used to store a "box" range on time series charts. [Learn more about time series chart ranges.](/docs/foundry/quiver/timeseries-ranges/)

## Input type

Flow start

## Output type

X/Y range
