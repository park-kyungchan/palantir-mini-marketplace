# List

Display a list of all objects in an object set.  Hovering over a list item opens a popover with object details, and allows opening the object in [Object Explorer](/docs/foundry/object-explorer/overview/).

## Input type

Object set

## Output type

Object selection

## Usage information

| Functionality | Availability |
| --- | --- |
| [Standard Quiver card](/docs/foundry/quiver/core-concepts/#cards) | Supported |
| [Transform table transform](/docs/foundry/quiver/cards-transform-table/) | Unsupported |
