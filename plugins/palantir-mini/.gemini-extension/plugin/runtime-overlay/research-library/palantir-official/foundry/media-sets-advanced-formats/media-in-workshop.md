# Using media in Workshop

There are lots of tools in Workshop for displaying and working with media in your applications.

Available widgets for media include:

* [Media preview](/docs/foundry/workshop/widgets-media-preview/)
* [Audio recorder](/docs/foundry/workshop/widgets-audio-recorder/)
* [Image annotation](/docs/foundry/workshop/widgets-image-annotation/)

Some media types have specialized widgets with advanced functionality. If your object type only leverages one media schema, you may consider using one of these specialized media widgets.

Specialized media widgets include:

* [PDF Viewer](/docs/foundry/workshop/widgets-pdf-viewer/)
* [Spreadsheet Display](/docs/foundry/workshop/widgets-spreadsheet-display/)
* [Video Display](/docs/foundry/workshop/widgets-video-preview/)
* [Audio and Transcription Display](/docs/foundry/workshop/widgets-audio-preview/)

To allow users to upload media via Workshop, you can set up an Action and configure the [button group widget](/docs/foundry/workshop/widgets-button-group/) to call this Action.
