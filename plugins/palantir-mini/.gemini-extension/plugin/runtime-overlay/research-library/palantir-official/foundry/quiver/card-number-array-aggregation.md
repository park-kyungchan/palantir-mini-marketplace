# Number array aggregation

Performs a single number aggregation on a number array.

The available aggregates are:

* Sum
* Average
* Standard deviation
* Maximum
* Minimum
* Difference
* Product
* Count
* First point
* Last point

## Input type

Number array

## Output type

Number
