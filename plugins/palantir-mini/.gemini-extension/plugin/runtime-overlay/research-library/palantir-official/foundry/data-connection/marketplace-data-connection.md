# Add sync to a Marketplace product

Use [Foundry DevOps](/docs/foundry/devops/overview/) to include your Data Connection syncs in [Marketplace products](/docs/foundry/devops/core-concepts/#product) for other users to install and reuse. [Learn how to create your first product.](/docs/foundry/foundry-devops/create-products/)

## Supported features

Both standard [syncs](/docs/foundry/data-connection/set-up-sync/) and [streaming syncs](/docs/foundry/data-connection/set-up-streaming-sync/) can be included in Marketplace products.

## Adding Data Connection syncs to products

To add a Data Connection sync to a product, first [create a product](/docs/foundry/foundry-devops/create-products/). [Add outputs](/docs/foundry/foundry-devops/create-products/#add-outputs), then select the **Data connection sync** output type.

Once you have selected a Data Connection source, you can select which syncs you would like to include in your product.
