# Date/time parameter

Create a date/time input field which can be used as a variable in other cards that take a time as input.

Examples include:

* Date and timestamp property filter values in the [filter object set](/docs/foundry/quiver/card-filter-object-set/) card.
* Input to [date markers](/docs/foundry/quiver/timeseries-visualize/#date-markers) on time series charts.
* Input to date markers on [overlay charts](/docs/foundry/quiver/card-overlay-chart/).

## Input type

Flow start

## Output type

Time
