# Customize Foundry Rules

The default configuration described in the [deploy Foundry Rules](/docs/foundry/foundry-rules/deploy-foundry-rules/) documentation is designed to apply generally to all Foundry Rules deployments. However, you may want to additionally tailor your Foundry Rules workflow to your specific use case.

These are some common ways to customize Foundry Rules:

* [Enable optional features](/docs/foundry/foundry-rules/enable-optional-features/)
* [Add a custom property](/docs/foundry/foundry-rules/add-a-custom-property/)
* [Set up permissions for editing rules](/docs/foundry/foundry-rules/rule-permissions/)
* [Permitted and default output values](/docs/foundry/foundry-rules/permitted-and-default-output-values/)
* [Customize your Foundry Rules pipeline](/docs/foundry/foundry-rules/customize-foundry-rules-pipeline/)
