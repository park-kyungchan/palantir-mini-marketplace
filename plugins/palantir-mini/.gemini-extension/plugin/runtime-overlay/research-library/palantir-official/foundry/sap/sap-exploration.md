# Source exploration

Once the [source has been created](/docs/foundry/sap/sap-source-setup/), the SDDI SAP Source Explorer can be used for interactive source exploration. Note that all data in the following screenshots is notional.

1. Navigate to the source and select the **Explore and create syncs** button.

![Explore SAP Source](/docs/resources/foundry/sap/sap-source-explore.png)

2. For remote and SLT connections, select the appropriate context.

![SAP Explorer Landing](/docs/resources/foundry/sap/sap-explorer-landing.png)

3. Select the tables or modules of interest and add them onto the graph.

![SAP Explorer Graph](/docs/resources/foundry/sap/sap-explorer-graph.png)

4. Proceed to [configuring the syncs](/docs/foundry/sap/sap-sync-setup/).
