# transforms.api.FloatParam

## *class* transforms.api.FloatParam(default, \*, description=None) {#transforms.api.FloatParam}

Specification for the `ParameterSpec` definition used as an input to a transform.

* **Parameters:**
  * **default** ([*float* ↗](https://docs.python.org/3/library/functions.html#float)) – The default value for the parameter.
  * **description** ([*str* ↗](https://docs.python.org/3/library/stdtypes.html#str) *,* *optional*) – The parameter description.

### input\_results(\_instance, \*\*\_kwargs) {#transforms.api.FloatParam.input\_results}

Generates a map from input RIDs to `TransformsInputResult`. Results are merged into a `TransformJobResult`.

* **Parameters:**
  **instance** – An object of the type generated by `instance()`.

### *static* instance(context, json\_value) {#transforms.api.FloatParam.instance}

Creates a parameter instance using the raw JSON value from JobSpec parameters and specific context.

The return value is injected in the transform compute function.

* **Parameters:**
  * **context** ([*ParamContext*](/docs/foundry/api-reference/transforms-python-library/api-paramcontext/#transforms.api.ParamContext)) – A context object with properties that might be required for creating an instance.
  * **json\_value** (*any*) – Any raw value deserialized from JobSpec parameters.

### *property* json\_value {#transforms.api.FloatParam.json\_value}

Returns the JSON value for this parameter to put in JobSpec.

If the return value is `None`, the parameter is considered unbound. If any transform’s parameter is unbound, the transform is considered to be unbound. For unbound transforms, the JobSpec is not published.

### *static* lightweight\_instance(context, json\_value) {#transforms.api.FloatParam.lightweight\_instance}

Instantiate an input type from the resolved JSON value.

### *property* schema {#transforms.api.FloatParam.schema}

Returns JSON schema for parameters of this type. Must return a valid JSON schema.
