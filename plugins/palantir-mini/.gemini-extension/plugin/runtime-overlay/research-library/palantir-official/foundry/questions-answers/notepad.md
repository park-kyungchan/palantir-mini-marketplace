# Notepad

### Is it possible to display properties of a linked object in Notepad through the Object Property or Object Card widget?

It is currently not possible to display properties of a linked object in a Notepad. However, in a Notepad template, you can configure a function to return the linked objects.

*Timestamp:* April 16, 2024

### Is embedding a variable in a notepad template header/footer supported?

Embedding a variable in a template header/footer is not supported directly at the moment, but widgets can be copied into the header/footer menu as a workaround.

*Timestamp:* April 16, 2024

### Is it possible to merge cells in the Table widget in Notepad?

Currently, it is not possible to merge cells in the Table widget.

*Timestamp:* April 16, 2024

### How do I turn on the table of contents next to the scroll bar in a document?

The table of contents can be enabled from the view menu.

*Timestamp:* April 13, 2024

### In Notepad, is it possible to pass a property of the object from a section generator as an input into a Quiver dashboard chart?

A property of the object cannot be passed. However, the entire object can be passed.

*Timestamp:* July 24, 2024

### Is it possible to add Notepads as documentation that can be leveraged by AIP Assist to answer questions?

Yes, it is possible to add Notepad documents as custom documentation that AIP Assist can reference when answering questions. You can add Notepad documents to AIP Assist with minimal configuration by using the "Add to AIP Assist" option from the "Actions" dropdown menu.

*Timestamp:* November 13, 2024
