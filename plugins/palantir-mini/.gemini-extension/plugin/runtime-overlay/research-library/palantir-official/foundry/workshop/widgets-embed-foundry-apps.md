# Embed Foundry apps

* **[Iframe](/docs/foundry/workshop/widgets-iframe/):** Embed a custom widget URL or Slate application via iframe.
* **[Derived series](/docs/foundry/time-series/derived-series-widget/):** Embed a derived series management space to create, update, and discover derived series.
* **[Map Application Template](/docs/foundry/map/widget/):** Display an existing geospatial template created in the Map application.
* **[Notepad - Embedded Document](/docs/foundry/notepad/workshop-embed/):** Embed a configured Notepad document in either read-only or edit mode.
* **[Quiver Dashboard](/docs/foundry/quiver/dashboards-workshop/):** Render an embedded dashboard constructed and published from Quiver.
* **[Vertex Graph](/docs/foundry/vertex/embed-graph-workshop/):** Display an existing graph, graph template, or diagram created in the Vertex application.
* **[Workshop: Embedded Module](/docs/foundry/workshop/embedded-modules/):** Embed another Workshop module as a widget.
