# Set Carbon as the Foundry landing page

In some cases, you will want users to immediately land on a Carbon workspace when accessing Foundry; in other words, for a Carbon workspace to be the user home page in Foundry. To configure this setting for your entire organization, go to `/workspace/control-panel` and navigate to **Platform experience** in the left-hand panel. If you wish for users to land on their default workspace, set the **Home page URL configuration** to `/carbon`.

If you wish to configure this on a more granular level and control which page specific groups of users land on, click **New home page URL**.
