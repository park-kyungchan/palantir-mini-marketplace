# Uppercase

> Supported in: Batch, Faster, Streaming

Converts all characters in string to uppercase.

**Expression categories:** String

## Declared arguments

* **Expression:** Expression to convert to uppercase.<br>*Expression\<String>*

**Output type:** *String*

## Examples

### Example 1: Base case

**Argument values:**

* **Expression:** hello World

**Output:** HELLO WORLD

***

### Example 2: Null case

**Argument values:**

* **Expression:** *null*

**Output:** *null*
