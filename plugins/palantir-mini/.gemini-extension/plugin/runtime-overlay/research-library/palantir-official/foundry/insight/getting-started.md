# Getting started

The following tutorial walks you through using Insight to analyze Ontology data, from finding and opening resources to saving and sharing results.

## Create a new analysis

To create a new Insight analysis, select **+ New analysis** from the top right of the home page.

![The new analysis button in the top right of the Insight home page.](/docs/resources/foundry/insight/new-analysis.png)

You can also create a new Insight analysis from other applications:

* From [Object Explorer](/docs/foundry/object-explorer/overview/), select any object type or object set to view an option to analyze the data in Insight.
* From other applications, an **Open in Insight** option is available from the **Actions** menu.

## Find data

When you create a new Insight analysis, you must first select Ontology data. Insight works with the Foundry Ontology, allowing you to search for object types, object sets, and [interfaces](/docs/foundry/interfaces/interface-overview/). Use the **Choose starting data** window to find data based on names and descriptions.

![The Choose starting data window to search the ontology for object types.](/docs/resources/foundry/insight/starting-data.png)

For more advanced searching, select the filter icon to the right of the search bar. Available filters include the following:

* [Statuses](/docs/foundry/object-link-types/metadata-statuses/)
* [Spaces](/docs/foundry/security/orgs-and-spaces/#spaces)
* [Object type groups](/docs/foundry/object-link-types/type-groups/)
* [Interfaces](/docs/foundry/interfaces/interface-overview/)
* [Shared properties](/docs/foundry/object-link-types/shared-property-overview/)

![Use the filter options to search by status, groups, and interfaces.](/docs/resources/foundry/insight/add-data-filter.png)

Filtering allows for more complex searching, using a combination of filters to reduce the search space to find a particular object type.

For example, you can select the `Endorsed` status filter and choose only certain groups to search within to significantly narrow the scope of results.

![A complex search using a combination of status and group filters to narrow the results.](/docs/resources/foundry/insight/complex-search.png)

## Preview and select data

Object types can be previewed for more detailed information by selecting a card in the search results. You can select multiple object types to open multiple previews. The preview provides information such as properties, groups, and linked object types. Individual objects can also be previewed.

![The preview panel shows properties, groups, and linked object types for a selected object type.](/docs/resources/foundry/insight/preview-objects.png)

Choose one or many object types and add them to the analysis. Select **Start analysis** in the bottom right corner, or select **Open** on any object type card to begin creating your Insight analysis.

Learn more about building an [analysis path](/docs/foundry/insight/analysis-panel/) for your data.
