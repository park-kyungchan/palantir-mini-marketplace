# Get PDF page dimensions

> Supported in: Batch, Faster

Get the dimensions in points of each page of the PDF.

**Expression categories:** Media

## Declared arguments

* **Media reference:** The document from which page dimensions are extracted.<br>*Expression\<Media reference>*

**Output type:** *Array\<Struct\<height:Double, width:Double>>*
