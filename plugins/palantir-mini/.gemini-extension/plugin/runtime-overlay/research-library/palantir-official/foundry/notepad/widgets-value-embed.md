# Value embed

:::callout{theme="neutral"}
This widget is only available for [document templates](/docs/foundry/notepad/templates-overview/).
:::

The **value embed** widget allows you to use template inputs to insert text directly into a document when it is generated from the template.

![notepad\_widgets\_value\_embed\_in\_template](/docs/resources/foundry/notepad/notepad_widgets_value_embed_in_template.png)

During the template generation process, the contents of the template input will be inserted into the document in place of the value embed widget. For example, the following document was generated from the template shown above:

![notepad\_widgets\_value\_embed\_generated](/docs/resources/foundry/notepad/notepad_widgets_value_embed_generated.png)

## Template configuration

* **Template input selection:** The template input whose value is to be inserted into the document.
