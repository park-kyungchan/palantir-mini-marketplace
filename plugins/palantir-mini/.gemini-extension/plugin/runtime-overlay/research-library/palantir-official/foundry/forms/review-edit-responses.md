# Review and edit responses

:::callout{theme="warning"}
Foundry Forms is no longer the recommended approach for data entry or writeback workflows on Foundry. Instead, build user input workflows with the Foundry Ontology, representing the relevant data structures as object types and configuring the writeback interaction with Actions. Learn more in the [Forms overview](/docs/foundry/forms/overview/) documentation.
:::

Users who follow a form will receive a [notification](/docs/foundry/forms/notifications/) after each submission. From there, users can review and edit responses by opening the form `results` spreadsheet.

To view and edit responses stored in Fusion spreadsheets, perform the following steps:

1. First, navigate to the form in Editor mode.
2. Select the **View entries** button to the left of the green **Save** button in the upper right of the screen.

* Alternatively, open the corresponding form `results` spreadsheet directly in Fusion.

3. Select a cell within a response.
4. Select the **Forms** tab at the top and choose **Open row**. Edits can now be made for this specific response.
5. Select the green **Save changes** button in the bottom right.
